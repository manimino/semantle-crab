# semantle-crab
Solve [Semantle](https://semantle.novalis.org/) / [Pimantle](https://semantle.pimanrul.es/) in 4 moves. 

Guess "giant", then "enemy", then "crab". 

[Then put numbers in here and hit solve.](https://semantle-crab.manimino.com/)
